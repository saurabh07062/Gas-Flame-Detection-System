
## Real-Time Gas Detection System
---

## 📌 Project Description

This project is a **Real-Time Gas and Flame Detection System** built using ESP32, MQ2 gas sensor, and a flame sensor. It continuously monitors gas levels and fire presence, sends data to a backend server, and displays it on a live web dashboard.

---

## 🎯 Objectives

* Detect gas leakage in real-time
* Detect fire/flame instantly
* Display sensor data on a dashboard
* Provide alerts for safety

---

## 🧰 Technologies Used

* **Hardware:** ESP32, MQ2 Gas Sensor, Flame Sensor
* **Backend:** Node.js (Express)
* **Frontend:** HTML, CSS, JavaScript
* **Communication:** HTTP / Wi-Fi

* 🌐 Wi-Fi based data transmission
* ⚡ Fast and efficient system

---

## 🔌 How It Works

1. ESP32 reads data from MQ2 gas sensor and flame sensor
2. Data is sent to backend server via Wi-Fi
3. Server processes and stores the data
4. Frontend dashboard displays real-time values

---

## 🚀 How to Run the Project

### 1️⃣ ESP32 Setup

* Upload the ESP32 code using Arduino IDE
* Connect MQ2 and Flame sensor properly

### 2️⃣ Backend Setup

```
cd backend
npm install
node server.js
```

### 3️⃣ Frontend Setup

* Open `index.html` in browser
